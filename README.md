#### Linux安装教程

 **V2** 

在lib目录下git本仓库

```
git clone https://gitee.com/ying_Sailor_uniform/more-js.git ./Temp
```
移动文件

```
mv Temp/more-js-V2.js example
```

 **V3** 

在plugins目录下git本仓库

移动文件

```
mv Temp/more-js-V3.js example
```

安装完成可以删除temp文件夹`rm -rf Temp`
### Windows安装教程
下载本仓库文件，将插件文件移至lib\example内即可

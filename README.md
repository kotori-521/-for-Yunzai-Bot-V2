#### Linux安装教程
在lib目录下git本仓库

```
git clone https://gitee.com/ying_Sailor_uniform/V2-js.git ./Temp
```
移动文件

```
mv Temp/more-js.js example
```
安装完成可以删除temp文件夹`rm -rf Temp`
### Windows安装教程
下载本仓库文件，将插件文件以至lib\example内即可
